# Mondrian Project

A recreation of Piet Mondrian's iconic artwork using HTML and CSS Grid.

## About This Project

This project was completed as part of Dr. Angela Yu's Complete Web Development Bootcamp. The goal was to recreate a famous artwork while practicing CSS Grid, positioning, spacing, and layout techniques.

## Technologies Used

* HTML5
* CSS3
* CSS Grid

## Features

* CSS Grid layout
* Precise positioning
* Responsive structure
